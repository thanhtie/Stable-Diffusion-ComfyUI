### FLUX.1 [schnell] – Model Artifacts

- **Base model (rectified flow transformer) – FLUX.1 [schnell]**  
  Download primary checkpoint from Hugging Face:  
  → https://huggingface.co/black-forest-labs/FLUX.1-schnell  

- **ComfyUI-compatible model package**  
  Use the Comfy-ready workflow/model files from:  
  → https://huggingface.co/Comfy-Org/flux1-schnell/tree/main  

- **Text encoders (CLIP-equivalent for FLUX)**  
  Dedicated text encoder weights for FLUX pipelines:  
  → https://huggingface.co/comfyanonymous/flux_text_encoders/tree/main  

- **VAE (latents ↔ RGB decoder/encoder)**  
  VAE component for FLUX.1 (used to map between latent space và pixel space):  
  → https://huggingface.co/black-forest-labs/FLUX.1-dev/tree/main/vae
